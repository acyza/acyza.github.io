# vistal stdio code安装和使用
## 安装
以Windows环境为例

1. 进入vscode官网 ☛ [官网链接](https://code.visualstudio.com/)
2. 点击Download for Windows下载
3. 打开安装程序
4. 下一步，下一步……
5. 安装完成

和普通的安装程序一样，非常简单

安装完成后不装插件拿来学习Web前端够用了
